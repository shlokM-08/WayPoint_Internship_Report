# Adden Internship Report

A React + TypeScript application displaying work hours with a shooting stars background effect.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Shooting stars animated background
- Work hours table with glassmorphism styling
- Responsive design
- TypeScript support
